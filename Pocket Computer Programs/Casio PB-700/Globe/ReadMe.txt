This is the globe plotting program originally published for the Sharp PC-1500 / TRS-80 PC-2.  It was converted a few times for other models, I have made minor updates for the PB-700 / PB-770.

For the prompts, I use these as an example:

R(MM) : 11
Alpha : 10
Beta  : 0
Phi   : -60
Grid  : 45
Step  : 5
Trans : N
Color : N
